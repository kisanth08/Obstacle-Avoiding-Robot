# Obstacle-Avoiding-Robot
Moving on its own and avoids obstacles in its path. The car uses an ESP32 microcontroller, an L298N motor driver, an ultrasonic sensor. When the sensor spots objects , &lt;25cm (car moves back and turns left) ,&lt;=10cm (car stops completely until its clear). The 12V power supply runs the motors, while ESP32 is powered by 5V.
